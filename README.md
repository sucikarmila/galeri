⚙️ Instalasi Cepat (Quick Start)
Clone Repositori:
git clone https://github.com/sucikarmila/galeri.git

Persiapan Database:
Buat database db_gallery di MySQL, lalu jalankan:
composer install -> php artisan key:generate

Migrasi:
php artisan migrate (Bebas error storage:link).

Jalankan:
php artisan serve (Akses di http://127.0.0.1:8000).
